# MVC-Pattern
Это очень простая реализация MVC. 
### Структура
```
Project
├── App
│   ├── Core
│   │   ├── App.php
│   │   └── DataBase
│   ├── Views
│   │   ├── errors.php
│   │   └── main.php
│   ├── Controller.php
│   ├── init.php
│   └── Model.php
└── Public
    ├── css
    │   └── style.css
    ├── js
    │   └── script.js
    └── index.php
```
